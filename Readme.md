## Creating a set.xml file ##

### Requirements ###
 * Python 2.7
 * Python Modules:
    datetime
    requests
    feedparser
    re
    json
    shutil
    sys
    urllib
    os
    zipfile


### Usage ###
    
```
$> python spoiler.py
```

Outputs {SETCODE}.json, {SETCODE}.xml, AllSets.json.zip, and Images.zip (containing {SETCODE}\{IMAGE NAMES}.jpg)

Create the set xml file and add it to your `customsets` folder for Cockatrice.
