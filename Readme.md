## Creating a set.xml file ##

### Requirements ###
 * Python 2.7
 * Python Modules:
    datetime
    requests
    feedparser
    re
    json
    shutil
    sys
    urllib
    os
    zipfile

```
pip install requests feedparser
```

### Usage ###
    
```
$> python spoilers.py
```

Outputs {SETCODE}.xml and MPS\_{SETCODE}.xml

Create the set xml file and add it to your `customsets` folder for Cockatrice.
