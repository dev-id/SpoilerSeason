## Creating a set.xml file ##

### Requirements ###
 * Python 2.7
 * python requests (pip install requests)
 * feedreader (pip install feedreader)
 * json
 * datetime

### Usage ###
    
```
$> python spoiler.py
```

Outputs EMN.json and EMN.xml

Create the set xml file and add it to your `customsets` folder for Cockatrice.
